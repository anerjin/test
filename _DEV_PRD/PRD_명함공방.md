## 1. 제품 개요

### 1.1 제품 비전
"명함공방"은 프리미엄 명함 전문 제작 업체의 온라인 홈페이지로, **원페이지 형태**로 회사 소개부터 명함 제작 접수까지 모든 과정을 한 페이지에서 완결하는 서비스입니다.

### 1.2 목표 사용자
- **1차 타겟**: 스타트업/중소기업 대표 및 임직원 (25~45세)
- **2차 타겟**: 중견기업 총무/관리부서 (대량 주문)
- **3차 타겟**: 자영업자, 프리랜서 (개인 명함)

### 1.3 핵심 가치 제안
> "당신의 첫인상을 디자인합니다" — 프리미엄 명함 전문 제작, 24시간 온라인 접수

## 2. 기능 명세 (Feature Specifications)

### F-001: 고정 네비게이션 바

**우선순위**: Must Have

| 항목 | 명세 |
|------|------|
| **설명** | 페이지 상단에 고정되는 네비게이션 바 |
| **데스크탑** | 로고 + 메뉴(서비스, 포트폴리오, 가격, 접수) + 다크모드 토글 |
| **모바일** | 로고 + 햄버거 메뉴 + 다크모드 토글 |
| **스크롤 동작** | 50px 이상 스크롤 시 배경 블러(16px) + 그림자 추가 |
| **활성 상태** | 현재 보이는 섹션에 해당하는 메뉴 항목 하단에 골드 언더라인 |
| **모바일 메뉴** | max-height 트랜지션으로 펼침/접힘, 메뉴 클릭 시 자동 닫힘 |
| **접근성** | aria-expanded, aria-label 동적 업데이트 |
| **성능** | requestAnimationFrame + passive scroll listener |

**인수 조건 (Acceptance Criteria)**:
- [ ] 스크롤 시 배경이 반투명+블러로 전환된다
- [ ] 현재 섹션에 해당하는 메뉴가 활성화된다
- [ ] 모바일에서 햄버거 메뉴가 정상 작동한다
- [ ] 키보드 탭으로 모든 메뉴에 접근 가능하다

### F-002: 히어로 섹션

**우선순위**: Must Have

| 항목 | 명세 |
|------|------|
| **배경** | 다크 그라디언트 (140deg, #0c0c1d → #1a1a2e → #16213e → #0f3460) |
| **장식 요소** | 도트 패턴, 글로우 그라디언트, 플로팅 쉐이프 (8s infinite) |
| **좌측 콘텐츠** | 메인 카피 + 서브 카피 + CTA 2개 |
| **우측 콘텐츠** | 명함 3D 목업 (perspective + hover rotate3d) |
| **CTA 1** | "무료 견적 받기" → #contact 스크롤, 골드 배경 + pulse 애니메이션 |
| **CTA 2** | "포트폴리오 보기" → #portfolio 스크롤, 아웃라인 스타일 |
| **통계 바** | 하단 4칸: 고객 15,000+, 제작 50,000+, 만족도 99.2%, 경력 12년+ |
| **통계 애니메이션** | IntersectionObserver 50% 감지 시 easeOutExpo 카운팅 (2200ms) |
| **모바일** | 세로 배치, 목업 축소, 통계 2x2 그리드 |

**인수 조건**:
- [ ] 히어로가 뷰포트 전체를 차지한다 (min-height: 100vh)
- [ ] CTA 클릭 시 해당 섹션으로 부드럽게 스크롤된다
- [ ] 통계 숫자가 스크롤 시 0부터 카운팅된다
- [ ] 모바일에서 텍스트 가독성이 확보된다

|------|------|
| **필드 - 이름** | text, 필수, 2자 이상 |

| **필드 - 명함 종류** | select (일반/프리미엄/특수), 필수 |
| **필드 - 수량** | select (100/200/300/500/1000/기타), 필수 |
| **필드 - 후가공** | checkbox 복수선택 (코팅/금박/은박/엠보싱/도무송) |
| **필드 - 요구사항** | textarea, 선택 |
| **필드 - 개인정보 동의** | checkbox, 필수 |
| **스팸 방지** | 허니팟 hidden 필드 |
| **유효성 검증** | blur 이벤트에서 검증, input 이벤트에서 에러 클리어 |
| **에러 표시** | 필드 하단 빨간색 메시지 + 테두리 빨간색 + shake 애니메이션 |
| **포커스 관리** | 제출 실패 시 첫 번째 에러 필드로 자동 포커스 |
| **제출 상태** | 버튼 비활성 + 스피너 + "접수 중..." 텍스트 |
| **성공 처리** | 성공 모달 표시 + 폼 리셋 |
| **실패 처리** | 에러 모달 표시 + 재시도 안내 |
| **폼 액션** | Formspree 등 외부 서비스 POST |

**인수 조건**:
- [ ] 모든 필수 필드가 미입력 시 에러 메시지가 표시된다
- [ ] 연락처/이메일 형식 오류 시 구체적 에러 메시지가 표시된다
- [ ] blur 시 검증, input 시 에러 클리어가 정상 작동한다
- [ ] 제출 시 로딩 → 성공/에러 모달이 표시된다
- [ ] 허니팟 필드 입력 시 제출이 차단된다
- [ ] 접근성: aria-invalid, aria-label이 적절히 설정된다
