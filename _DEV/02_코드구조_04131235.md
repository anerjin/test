## 3. `js/main.js` 모듈 맵

### 3.1 초기화 플로우

```
DOMContentLoaded
└── initApp()
    ├── initTheme()              ← 다크모드 토글 + 시스템 동기화
    ├── initNavigation()         ← 스크롤 감지 + 활성 섹션
    ├── initMobileMenu()         ← 햄버거 메뉴
    ├── initSmoothScroll()       ← 앵커 스크롤
    ├── initScrollReveal()       ← IntersectionObserver fade-up
    ├── initStatsCounter()       ← 카운터 애니메이션
    ├── initPortfolioFilter()    ← 필터 + 스태거
    ├── initLightbox()           ← 포트폴리오 확대
    ├── initCarousel()           ← 후기 캐러셀
    ├── initFaqAccordion()       ← FAQ 토글
    ├── initContactForm()        ← 유효성 + 제출
    ├── initEstimator()          ← 실시간 견적
    ├── initBackToTop()          ← 플로팅 버튼
    └── initIcons()              ← lucide.createIcons()
```

### 3.2 주요 함수 시그니처

```js
// 테마
function initTheme()                              // 토글, 시스템 감지, localStorage 동기화
function applyTheme(theme: 'light'|'dark')         // data-theme 속성 + theme-color 업데이트

// 네비
function initNavigation()                          // scroll throttle + section detection
function updateActiveLink(sectionId: string)      // nav-link 활성화

// 폼
function validateField(field: HTMLInputElement): ValidationResult
function showError(field, message)
function clearError(field)
function calculateEstimate(): number               // base + add-ons - discount
function updateEstimateUI(amount: number)

// 캐러셀
function initCarousel(el: HTMLElement)
function goToSlide(index: number)
function nextSlide() / prevSlide()
function startAutoplay() / stopAutoplay()

// 유틸
function throttle(fn, delay)
function debounce(fn, delay)
function formatKRW(n: number): string             // ₩15,000
```

### 3.3 이벤트 바인딩 표

| 요소 | 이벤트 | 핸들러 | 리스너 옵션 |
|------|--------|--------|-------------|
| `window` | `scroll` | `handleScroll` | `{ passive: true }` |
| `window` | `resize` | `debounce(handleResize, 200)` | - |
| `#theme-toggle` | `click` | `toggleTheme` | - |
| `#mobile-menu-toggle` | `click` | `toggleMobileMenu` | - |
| `.nav-link[href^="#"]` | `click` | `smoothScrollTo` | - |
| `.form-input` | `blur` | `validateField` | - |
| `.form-input` | `input` | `clearError` | - |
| `#contact-form` | `submit` | `handleSubmit` | - |
| `.filter-chip` | `click` | `applyFilter` | - |
| `.portfolio-item` | `click` | `openLightbox` | - |
| `document` | `keydown (Esc)` | `closeLightbox / closeModal` | - |
| `.carousel` | `touchstart/move/end` | `swipeHandler` | `{ passive: true }` |

### 3.4 에러 처리 원칙

1. **Fail gracefully**: 외부 의존성(Formspree) 실패 시 모달로 사용자 안내
2. **콘솔 로그 금지**: 프로덕션 코드에서 `console.*` 제거
3. **Try/catch 범위**: 폼 제출, localStorage 접근, JSON 파싱 등 외부 경계
4. **네트워크 오류**: `response.ok` 체크 + `catch` 블록 모두 처리

```js
try {
  const res = await fetch(FORM_ENDPOINT, { method: 'POST', ... });
  if (!res.ok) throw new Error('Network response was not ok');
  showSuccessModal();
} catch (err) {
  showErrorModal();
} finally {
  setLoading(false);
}
```

## 4. CSS 변수 사용 가이드

### 4.1 권장 패턴

```css
/* ✅ Good: 시맨틱 토큰 사용 */
.card {
  background: var(--color-surface);
  color: var(--color-text);
  border: 1px solid var(--color-border);
}

/* ❌ Bad: 하드코딩 */
.card {
  background: #f8f9fa;
  color: #1a1a2e;
}
```

### 4.2 다크모드 오버라이드

추가 토큰 정의 시 반드시 `[data-theme="dark"]` 블록에도 대응값을 제공.

```css
:root {
  --color-new-accent: #ff6b6b;
}
[data-theme="dark"] {
  --color-new-accent: #ffa8a8;  /* 다크 대비 조정 */
}
```

## 6. 데이터 흐름 (견적 계산기 예시)

```

       ↓

  - DOM 업데이트 (#estimate-amount)
```
