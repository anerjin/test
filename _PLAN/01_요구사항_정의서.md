# 명함공방 - 요구사항 정의서 (Requirements Document)

> **문서 버전**: v1.0

> **작성자**: 프로덕트 기획팀

## 1. 프로젝트 개요

### 1.1 비즈니스 목표
프리미엄 명함 전문 제작 업체 "명함공방"의 온라인 원페이지 홈페이지를 통해 브랜드 신뢰도를 구축하고, 24시간 온라인 접수 체계를 마련하여 고객 전환율을 극대화한다.

### 1.2 KPI (핵심 성과 지표)

| KPI | 목표치 | 측정 방법 |
|-----|--------|-----------|
| 월간 방문자 수 | 3,000+ | Google Analytics |
| 문의/접수 전환율 | 5% 이상 | 폼 제출 수 / 방문자 수 |
| 평균 체류 시간 | 2분 이상 | Google Analytics |
| 모바일 이탈률 | 40% 미만 | Google Analytics |
| 페이지 로드 시간 | 3초 이내 | Lighthouse |

### 1.3 프로젝트 범위

- **In Scope**: 원페이지 반응형 웹사이트 (회사소개, 서비스, 포트폴리오, 가격, 접수폼, FAQ)
- **Out of Scope**: 회원가입/로그인, 결제 시스템, 관리자 대시보드, 주문 추적

## 4. 기술 스택

| 구분 | 기술 | 선정 사유 |
|------|------|-----------|
| 마크업 | HTML5 (Semantic) | SEO + 접근성 |
| 스타일링 | Tailwind CSS (CDN) | 빠른 프로토타이핑, 반응형 |
| CSS 커스텀 | CSS Variables + custom.css | 다크모드 토큰 시스템 |
| 아이콘 | Lucide Icons (SVG) | 가볍고 일관된 아이콘 |
| 폰트 | Pretendard Variable | 한글 최적화, 가변 폰트 |
| JavaScript | Vanilla JS (ES5+) | 의존성 없음, 빠른 로딩 |
| 폼 백엔드 | Formspree / 외부 API | 서버리스 폼 처리 |
| 호스팅 | GitHub Pages | 무료, CI/CD 자동 배포 |
| 분석 | Google Analytics 4 | 트래픽/전환 추적 |

## 5. 제약 조건

1. **예산**: 외부 API 비용 최소화 (무료 티어 우선)
2. **인력**: 프론트엔드 1인 개발, 서버 없음
3. **기한**: MVP 1주 내 완성
4. **콘텐츠**: 실제 포트폴리오 이미지는 추후 교체 (플레이스홀더 사용)
5. **폼 제출**: 서버리스 방식 (Formspree 등 외부 서비스 활용)
