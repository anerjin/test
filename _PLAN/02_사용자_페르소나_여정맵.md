## 3. 핵심 사용자 시나리오

### 시나리오 1: 온라인 접수 (Happy Path)

```
1. 사용자가 검색을 통해 명함공방 웹사이트에 접속한다
2. 히어로 섹션에서 "무료 견적 받기" CTA를 클릭한다
3. 접수 폼 섹션으로 스무스 스크롤 된다
4. 이름, 연락처, 이메일을 입력한다 (실시간 유효성 검증)
5. 명함 종류 "프리미엄"을 선택한다
6. 수량 "200매"를 선택한다
7. 후가공 옵션에서 "금박"을 체크한다
8. 실시간 예상 견적 "60,000원"이 표시된다
9. 요구사항에 "로고 포함, 양면 인쇄" 작성한다
10. 개인정보 수집 동의에 체크한다
11. "접수하기" 버튼을 클릭한다
12. 로딩 → 성공 모달이 표시된다
13. "확인 이메일이 발송되었습니다" 안내
```

### 시나리오 2: 모바일 정보 탐색 후 카카오 문의

```
1. 사용자가 모바일에서 사이트에 접속한다
2. 햄버거 메뉴를 열어 "포트폴리오"를 탭한다
3. 포트폴리오 이미지를 확인하고 라이트박스로 상세 보기
4. 가격표를 확인한다
5. 우측 하단 카카오톡 플로팅 버튼을 탭한다
6. 카카오톡 채팅으로 이동하여 상담 시작
```

### 시나리오 3: 폼 입력 오류 처리

```
1. 사용자가 접수 폼에서 연락처에 잘못된 형식 입력
2. 포커스 아웃 시 "올바른 연락처를 입력해주세요" 에러 표시
3. 사용자가 올바른 형식으로 수정
4. 에러 메시지가 사라지고 정상 처리
5. 필수 항목 미입력 시 제출 버튼 클릭 → 첫 번째 에러 필드로 포커스 이동
```
