> **문서 버전**: v2.0
> **작성일**: 2026-04-12

## 1. 디자인 컨셉

### 1.1 스타일: Premium Minimalism + Gold Accent

명함공방은 **프리미엄 미니멀리즘** 스타일을 채택합니다. 비즈니스 인쇄 서비스의 신뢰감과 고급스러움을 동시에 전달하며, 골드 액센트로 프리미엄 품격을 표현합니다.

### 1.2 디자인 원칙

| 원칙 | 설명 |
|------|------|
| **고급스러움 (Luxury)** | 다크 네이비 + 골드 액센트로 프리미엄 인상 |
| **신뢰감 (Trust)** | 깔끔한 레이아웃, 충분한 여백, 통계 데이터 |
| **직관성 (Clarity)** | 한눈에 파악 가능한 서비스/가격 구조 |
| **행동 유도 (Action)** | 모든 섹션에서 접수/문의로 연결되는 CTA |

### 1.3 안티 패턴 (절대 사용 금지)

- 이모지를 아이콘으로 사용하지 않음 (Lucide SVG 아이콘만 사용)
- 무채색 단독 조합 (gray-on-gray) 금지
- 과도한 그림자 또는 3D 이펙트 남용 금지
- 플랫 디자인과 스큐어모피즘 혼용 금지
- 장식적 목적만의 애니메이션 금지

### 7.1 이징 커브

| 이름 | 값 | 용도 |
|------|-----|------|
| ease-out-expo | `cubic-bezier(0.16, 1, 0.3, 1)` | 등장 애니메이션 |
| ease-spring | `cubic-bezier(0.34, 1.56, 0.64, 1)` | 바운스 효과 |
| ease (CSS 기본) | `ease` | 일반 전환 |

### 7.2 지속 시간

| 토큰 | 값 | 용도 |
|------|-----|------|
| duration-fast | 150ms | 호버, 포커스 |
| duration-normal | 300ms | 일반 전환 |
| duration-slow | 500ms | 배경색 전환 |

### 7.3 애니메이션 패턴

| 이름 | 동작 | 시간 |
|------|------|------|
| fade-up | opacity 0→1, translateY 40→0 | 700ms |
| fade-in | opacity 0→1 | 700ms |
| fade-left | opacity 0→1, translateX -40→0 | 700ms |
| fade-right | opacity 0→1, translateX 40→0 | 700ms |
| scale-in | opacity 0→1, scale 0.92→1 | 600ms |
| stagger-children | 자식 요소 순차 등장 | 80ms 간격 |
| float-gentle | 3D 떠다니는 효과 | 8s infinite |
| pulse-gold | 골드 박스 쉐도우 펄스 | 2.5s infinite |
| price-pop | scale 1→1.08→1 | 400ms |
| shake | translateX 진동 | 400ms |
| spin | 360도 회전 | 600ms linear |

### 7.4 접근성: 모션 감소

```css
@media (prefers-reduced-motion: reduce) {
  * { animation-duration: 0.01ms !important; transition-duration: 0.01ms !important; }
  .fade-up, .fade-in, .scale-in { opacity: 1; transform: none; }
}
```

## 8. 아이콘 시스템

### 8.1 아이콘 라이브러리

- **라이브러리**: Lucide Icons (SVG)
- **로딩**: unpkg CDN
- **사이즈**: 기본 24px, 소형 16~20px, 대형 32~48px
- **스트로크**: 기본 2px (일관된 스타일)
