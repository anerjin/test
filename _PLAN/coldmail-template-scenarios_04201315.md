## 템플릿 3: 포트폴리오 공유

### 기본 정보

| 항목 | 내용 |
|------|------|

| **목표 전환율** | 메일 오픈 25%+ / 포트폴리오 페이지 방문 5%+ |

### 주요 섹션 구성

```
┌─────────────────────────────────────┐
│  [1] 헤더                            │
│  - 로고 + "Our Recent Work"          │
├─────────────────────────────────────┤
│  [2] 도입부                          │
│  - "최근 프로젝트를 공유드립니다"      │
│  - 짧은 맥락 설명 (1-2줄)            │
├─────────────────────────────────────┤
│  [3] 프로젝트 카드 (2-3개)           │
│  ┌─────────────────────────────┐    │
│  │ [썸네일 이미지]              │    │
│  │ 프로젝트명                   │    │
│  │ 클라이언트 / 기간            │    │
│  │ 핵심 성과 한 줄              │    │
│  │ "자세히 보기 →"              │    │
│  └─────────────────────────────┘    │
│  - 각 카드: #FFFFFF, border, 12px   │
│  - 카드 간 간격: 24px               │
├─────────────────────────────────────┤
│  [4] 전체 포트폴리오 CTA             │
│  - "전체 포트폴리오 보기" 버튼        │
├─────────────────────────────────────┤
│  [5] 신뢰 배지 (선택)               │
│  - "N개 프로젝트 완료" 등 수치        │
├─────────────────────────────────────┤
│  [6] 푸터                            │
└─────────────────────────────────────┘
```

### CTA 문구

| 유형 | 문구 | 링크 대상 |
|------|------|-----------|
| **Inline (각 카드)** | "자세히 보기 →" | 개별 프로젝트 상세 페이지 |
| **Primary** | "전체 포트폴리오 둘러보기 →" | 포트폴리오 메인 페이지 |
| **Secondary** (선택) | "비슷한 프로젝트 상담하기" | 미팅 예약 또는 문의 폼 |

### 톤앤매너

| 항목 | 가이드 |
|------|--------|
| **격식도** | Professional-Casual (중간~약간 캐주얼) |
| **화법** | 공유형. "~드립니다", "~해보세요" |
| **핵심 톤** | 겸손하게 실적을 보여주는. 자랑이 아닌 공유의 느낌 |
| **문장 예시** | "최근 함께한 프로젝트들을 공유드립니다. 귀사에도 영감이 되기를 바랍니다." |

## 템플릿 5: 후속 팔로업 메일

| 항목 | 내용 |

```

│  - "지난 [날짜] 보내드린 메일의       │
│    후속으로 연락드립니다"              │
│  - 이전 메일 핵심 1-2줄 요약          │
├─────────────────────────────────────┤
│  [3] 추가 가치 제공 (선택)           │
│  ┌─────────────────────────────┐    │
│  │ 추가 자료 또는 인사이트       │    │
│  │ "최근 관련 케이스를 정리해     │    │
│  │  보았습니다"                   │    │
│  └─────────────────────────────┘    │
│  - 단순 리마인드가 아닌 새 가치 추가  │
├─────────────────────────────────────┤
│  [4] CTA 영역                        │
│  - 부담 없는 톤의 행동 유도           │
├─────────────────────────────────────┤
│  [5] 푸터 (간소화)                   │
└─────────────────────────────────────┘
```

### CTA 문구

### 톤앤매너

| 항목 | 가이드 |
|------|--------|
| **격식도** | Professional-Soft (부드러운 격식) |

| **문장 예시** | "바쁘신 줄 알지만, 혹시 검토하실 기회가 있으셨는지 여쭤봅니다." |

| 6 | 프로젝트 제안서 | 미팅 후 1-2일 | 미팅 완료 후 |
| 7 | 가격/견적 | 제안서 검토 후 | 견적 요청 시 |
| 8 | 감사 메일 | 각 마일스톤 직후 | 미팅/계약/프로젝트 완료 시 |

## 디자인 변형 규칙

모든 템플릿은 동일한 디자인 시스템을 공유하되, 용도에 따라 아래와 같이 변형합니다.

| 변형 요소 | 규칙 |
|-----------|------|
| **헤더 크기** | 제안서/이벤트 → 넓은 헤더 / 팔로업/감사 → 심플 헤더 |
| **카드 수** | 서비스소개/이벤트 → 3-4개 / 팔로업/감사 → 0-1개 |
| **CTA 수** | 견적/제안서 → Primary + Secondary + Tertiary / 팔로업/감사 → Primary만 |
| **메일 길이** | 서비스소개/이벤트/견적 → 길게 / 팔로업/미팅요청 → 짧게 |
| **개인화 수준** | 제안서/견적 → 높음 (고객사명 필수) / 서비스소개/이벤트 → 낮음 (범용) |
| **다크 헤더 사용** | 이벤트/서비스소개 → 사용 / 팔로업/감사 → 미사용 |
